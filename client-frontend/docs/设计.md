前端划分为 8大功能模块 + 1个共享层，严格对应概要文档中的用户旅程。
1. Auth & Multi-Company Workspace（认证与多公司切换）

登录 / 注册 / OAuth（Google / GitHub / 企业SSO）
公司列表页（Workspace Selector）
创建新公司向导（名称 + 行业 → 智能推荐结构）
公司切换器（顶部导航栏，类似Notion）
公司归档 / 暂停 / 删除（带二次确认）

2. Dashboard（公司仪表盘 - 用户最常驻页面）

概览卡片：任务完成率、预算使用率、Agent活跃度、最近活动
实时KPI图表（Recharts 或 Tremor）
CEO主动建议卡片（流式更新）
进行中任务列表 + 风险预警
快捷操作：下达新战略、查看Heartbeat总结、打开主群聊

3. Organization Studio（组织结构可视化与编辑器 - 核心差异化功能）

React Flow 主画布：层级树（Board → CEO → Dept → Agent）
节点类型：Board、CEO、Department、Agent（不同颜色/图标/形状）
拖拽调整汇报线、添加/删除/重命名节点
侧边栏节点属性编辑器（System Prompt、性格、模型、预算上限等）
变更历史快照对比（时间旅行式审计）
一键应用行业模板结构

4. Agent Marketplace & Management（Agent 招聘与管理）

Agent商城浏览 & 购买/订阅
部门内Agent列表（卡片/表格）
Agent详情页：角色、专长、Skills、记忆概览、消耗统计
批量招聘、模板招聘
Agent性格与Prompt可视化配置（类似Character.AI + 系统提示编辑器）

5. Collaboration Hub（实时协作与群聊 - 日常核心）

主群聊（用户 + CEO 默认常驻）
动态房间：部门群、任务群、临时讨论群
富文本 + Markdown + 流式回复 + @提及 + 回复引用
Human-in-the-loop审批流（卡片式审批按钮）
消息智能总结 & 搜索
在线成员头像列表 + 正在输入提示
语音消息（未来占位）

6. Tasks & Autonomy Center（任务与自治运行）

任务列表（公司级 / 部门级 / Agent级）
大目标输入 → CEO智能拆解预览
任务详情 + 执行日志 + 子任务树
Heartbeat历史记录与总结
进度甘特图 / Kanban视图（可选）

7. Memory & Knowledge Base（记忆与知识库）

全局搜索（公司记忆）
分层记忆视图：公司级 / 部门级 / Agent级
文档上传 & 自动摄入进度
RAG检索测试界面（调试用 + 用户可见）
知识提炼历史

8. Billing & Governance（费用与治理中心）

实时费用仪表盘 + 趋势图
预算设置（公司/部门/Agent三级）
消耗明细（按Agent/任务/Skill/模型维度）
模型路由策略配置
审计日志查询
安全审批设置

共享层（Shared / Core）

Layout：响应式侧边栏 + 顶部导航（公司切换 + 用户头像）
Modals系统：统一模态管理（create company, edit node, approve request等）
Hooks层：useCompany, useCurrentAgent, useCollaboration, useBilling等
Components：AgentCard, NodeCard, ChatMessage, FlowNode等可复用组件
Utils & Types：从 contracts 同步生成的TypeScript类型（推荐用openapi-typescript或tRPC风格）
