# 5. 前端 — 管理员与平台运维

对应后端需求：[`../../docs/requirements/05-管理员与平台运维.md`](../../docs/requirements/05-管理员与平台运维.md)。

> **说明**：管理面是否做成独立子应用或同构路由由架构决定；以下需求适用于「在浏览器中操作 `/api/admin/*`」的控制台场景。

## 5.1 动态路由治理

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-ADM-RT-01 | **路由规则列表** | 表格 + 增删改；复杂匹配条件用表单或 JSON 编辑器（需校验） | 错误配置提交前校验；后端错误可读 |
| FE-ADM-RT-02 | **刷新缓存** | 按钮触发 `POST .../refresh`；成功 toast | 运维知悉已生效 |

## 5.2 API Key

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-ADM-KEY-01 | **密钥列表与创建** | 创建后**仅一次**完整展示密钥；复制按钮；提示保存 | 关闭弹窗后无法再次查看明文（与后端一致） |
| FE-ADM-KEY-02 | **轮换** | 确认后调用 rotate；旧 Key 失效说明 | 列表状态更新 |

## 5.3 IP 访问控制

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-ADM-IP-01 | **白名单/黑名单** | CIDR 或单 IP 输入校验；列表增删 | 误配时有格式错误提示 |

## 5.4 审计

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-ADM-AUD-01 | **审计日志查询** | 筛选 + 分页 + 详情展开；敏感字段脱敏展示 | 与后端脱敏策略一致 |

## 5.5 可观测性

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-OPS-MET-01 | **指标**（可选） | 嵌入 Grafana 或展示原始 Prometheus 文本的只读视图 | 非管理员不可见 |
| FE-OPS-HL-01 | **健康检查** | 各依赖状态一览（若聚合接口存在）；否则外链文档 | 运维可快速判断 |

## 5.6 安全能力（集成方控制台）

若产品面向「配置签名与防重放」：

| 需求 ID | 描述 | 验收要点 |
|---------|------|----------|
| FE-SEC-01 | 展示当前环境是否要求 HMAC、示例 Header | 与网关文档一致 |
| FE-SEC-02 | 提供时间戳/nonce 说明与测试请求工具（可选） | 非法签名在工具中可复现错误 |
