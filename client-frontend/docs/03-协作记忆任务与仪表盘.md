# 3. 前端 — 协作、记忆、任务与仪表盘

对应后端需求：[`../../docs/requirements/03-终端用户-协作记忆任务与仪表盘.md`](../../docs/requirements/03-终端用户-协作记忆任务与仪表盘.md)。

## 3.1 协作（Collaboration）

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-COL-01 | **房间列表与主房间** | 列表高亮主房间；进入房间前校验权限 | 仅本租户房间可见 |
| FE-COL-02 | **消息区** | 时间线或会话流；发送消息；上拉/下拉分页加载历史 | 顺序正确；发送中防重复提交 |
| FE-COL-03 | **消息搜索** | 搜索框 + 结果列表；长请求（约 15s）显示 loading | 无结果空态 |
| FE-COL-04 | **成员管理** | 成员列表；添加/移除；角色展示（若 API 返回） | 权限不足时操作不可用 |
| FE-COL-05 | **房间摘要** | 触发后异步或同步 loading；失败可重试 | 用户理解可能耗时 |
| FE-COL-06 | **从组织节点批量加人** | 组织节点选择器 + 确认 | 与组织数据一致 |
| FE-COL-07 | **创建部门房间** | 表单创建；成功后出现在列表 | — |
| FE-COL-08 | **协作侧组织节点搜索** | 与主组织模块 UI 区分标签或面包屑，避免用户混淆 | — |

### 3.1.1 实时通道（WebSocket）

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-COL-WS-01 | **Socket 连接** | 登录后建立连接；断线显示「连接已断开」与**自动重连**（指数退避可配置） | 令牌失效时断开并提示重新登录 |
| FE-COL-WS-02 | **房间事件** | 新消息、成员变动等实时刷新 UI；与 REST 列表互补 | 无重复消息或冲突时可接受策略（以产品为准） |

## 3.2 记忆与知识（Memory / RAG）

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-MEM-01 | **写入记忆** | 表单或 API 调试式界面；大负载显示进度（约 30s 超时） | 成功/失败明确 |
| FE-MEM-02 | **语义检索** | 查询框 + 结果列表（片段、得分若返回） | 空结果提示 |
| FE-MEM-03 | **摘要** | 长 loading（约 60s）；可取消（若支持 AbortController） | — |
| FE-MEM-04 | **文档导入** | 文件选择 + 同步/异步路径；异步时展示任务 ID 或轮询状态（若后端暴露） | 大文件走异步提示 |

## 3.3 任务（Tasks）

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-TASK-01 | **任务列表/详情/编辑** | 看板或表格 + 详情抽屉；状态标签 | CRUD 完整；租户隔离 |
| FE-TASK-02 | **任务拆解** | 触发长请求；结果展示子任务树或可编辑列表 | 失败可重试 |
| FE-TASK-03 | **任务树与执行日志** | 树视图 + 日志时间线 | 日志追加顺序可读 |
| FE-TASK-04 | **分配与进度** | 指派人选择器（用户/Agent 以实现为准）；进度条或百分比 | 权限校验失败有提示 |

## 3.4 仪表盘（Dashboard）

| 需求 ID | 界面/场景 | 交互与状态 | 验收要点 |
|---------|-----------|------------|----------|
| FE-DASH-01 | **公司摘要** | 卡片/图表展示 `GET /v1/dashboard` 数据；加载与错误态 | 与业务模块数据无明显矛盾 |
| FE-DASH-02 | **计费摘要** | 与账单模块联动展示；可跳转明细 | 数字可对账 |
