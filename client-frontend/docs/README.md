# Foundry 客户端前端需求文档（系列索引）

本目录从**终端用户与管理员在浏览器中的体验**角度描述「前端应提供哪些界面与交互」，并与后端 [`../../docs/requirements/`](../../docs/requirements/) 中的用户功能需求一一对应，便于产品评审、设计与验收。

## 与其它文档的关系

| 文档 | 用途 |
|------|------|
| [`../../docs/requirements/`](../../docs/requirements/) | 用户功能需求（API、角色、验收要点） |
| [`../../docs/architecture/implemented-features.md`](../../docs/architecture/implemented-features.md) | 实现侧路由与能力列举 |
| **本目录 `client-frontend/docs/*.md`** | 前端：信息架构、页面、交互状态、无障碍与性能等非功能要求 |
